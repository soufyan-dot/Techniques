Getting Started

Installation

To add the bot to your server, visit the Add Bot page and follow the instructions.

Configuration

Use the bot's dashboard or commands to configure it according to your server's needs.

Commands

  • !help - Displays a list of all available commands.
  • !play [song] - Plays a song in the voice channel.
  • !ban [user] - Bans a user from the server.
  • !stats - Displays server and bot usage statistics.
